v0.2 note:
Normally you do NOT need to manually put scrcpy-server here.
Gradle task :app:ensureScrcpyServer downloads the official scrcpy-server-v4.0 automatically before build.

Manual fallback:
Download official release asset:
https://github.com/Genymobile/scrcpy/releases/download/v4.0/scrcpy-server-v4.0
Rename it to:
scrcpy-server-v4.0.jar
Put it in this directory.
